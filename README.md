# 🎭 Impostor

Jogo de dedução e festa para jogar em grupo passando o celular. Funciona 100% no
navegador, sem instalar nada — ideal para celular.

## ▶️ Como jogar

Abra o `index.html` no navegador (ou acesse o link publicado). Escolha a quantidade
de jogadores (3 a 10), digite os nomes e selecione um dos modos de jogo.

## 🎮 Modos de jogo

- **🃏 Palavras · impostor sabe** — todos recebem a mesma palavra; o impostor recebe
  uma palavra relacionada e é avisado que é o impostor.
- **🎭 Palavras · impostor não sabe** — igual ao anterior, mas o impostor não é avisado.
- **🗨️ Qual é a frase?** — todos recebem uma pergunta; o impostor recebe uma parecida.
  Tem o botão "Mostrar frase padrão" na discussão.
- **🐝 Soletrando** — competição de soletração com níveis de dificuldade (fácil, médio,
  difícil), eliminação direta ou sistema de vidas, ranking e tela de vencedor.
- **🎨 Desenhe a Palavra** — cada um desenha sua palavra e depois todos votam para
  descobrir o impostor. Configurável: impostor sabe / não sabe e eliminação por votação.

## 🗂️ Estrutura do projeto

| Arquivo | Conteúdo |
|---|---|
| `index.html` | Estrutura HTML (todas as telas) |
| `styles.css` | Estilos, cores e animações |
| `script.js` | Toda a lógica do jogo |

## 🛠️ Tecnologias

HTML, CSS e JavaScript puro — sem dependências e sem build. Os três arquivos precisam
ficar na mesma pasta.
